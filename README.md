# [aiohttp](https://github.com/aio-libs/aiohttp) practice

## 1. Setup virtualenv and install aiohttp module

aiohttp requires Python 3.5 or newer.

```bash
$ mkdir aiohttp-practice
$ cd aiohttp-practice
$ python -m venv venv
$ pip install aiohttp
```
